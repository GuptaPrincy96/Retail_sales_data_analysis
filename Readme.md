# Retail Sales Dashboard (Power BI)

## Project Overview
This Power BI dashboard provides insights into retail sales performance using a cleaned retail sales dataset. The report enables users to analyze revenue, profit, cost, product performance, regional performance, and sales trends through interactive visualizations and slicers.

## Objectives
* Monitor overall business performance.
* Analyze revenue, profit, and cost metrics.
* Identify top-performing products.
* Compare sales performance across regions.
* Track sales trends over time.
* Enable interactive filtering for detailed analysis.

## Dataset
* **Dataset Name:** retail_sales_data_dirty

## Dashboard Components
* **KPI Cards:** Total Revenue
* **Revenue by Product:** Donut Chart
* **Sales Trend Over Time:** Line Chart
* **Regional Profit Distribution:** 100% Stacked Column Chart
* **Average Units Sold:** Gauge Chart
* **Product Cost vs Profit:** Clustered Column Chart

## Interactive Filters
* Region Slicer
* Product Slicer
* Date Slicer

## Tools Used
* Power BI Desktop
* Power Query
* DAX
* Data Modeling
* Interactive Visualizations
